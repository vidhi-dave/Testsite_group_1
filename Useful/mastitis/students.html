One way to edit each existing webpage 'live'...

1. Open each page in your browser, copy the URL address.
2. Open Netscape Composer (part of the free Netscape Communicator suite).
3. In Composer, File - Open Page.
4. Paste URL into text box.
5. You can then edit the live html onscreen. Use RED colour text, to differentiate it from existing black text.
6. To 'strike out' the existing text that you wish to replace, without deleting it off screen, use Format - Style - Strikethrough.
7. Create a new folder and save each file into it as its existing name (to preserve current links).
