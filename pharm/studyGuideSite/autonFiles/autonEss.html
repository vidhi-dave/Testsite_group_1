Autonomic essays

1. Atropine is widely used in veterinary practice. Is it overused?

2. Draw an annotated diagram of a sympathetic postganglionic nerve terminal, giving examples of named drugs which interfere with the synthesis, storage, release, actions and inactivation of the transmitter noradrenaline

3. Drugs A and B are two neuromuscular blocking drugs. When administered intravenously to a cat drug A induces fasciculations prior to flaccid muscle paralysis, whereas drug B induces only flaccid paralysis. Describe the modes of action of drugs A and B naming an example of each drug type

4. With reference to named drugs discuss how you would demonstrate how the autonomic nervous system influences the eye. Indicate the potential therapeutic application of these drugs.
