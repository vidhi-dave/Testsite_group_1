Sedative drugs

Acepromazine is very unlikely to provide adequate sedation in an excited cat.

Detomidine would work but only comes in a concentration suitable for horses so accurate dosing is difficult. Use medetomidine instead.

Diazepam would not sedate this cat: it might make the cat even more excited.

Halothane could be used to anaesthetise the cat, but is likely to provoke potentially fatal arrhythmias in a very excited cat. This technique would only be used as a last resort.

Ketamine is probably the best choice.

Ketamine and diazepam is overkill. The only reason for adding the diazepam is to produce muscle relaxation, which is not necessary in this case.

Medetomidine would probably work but can cause severe cardiovascular depression.

Saffan sedation requires high volumes when given im, so it is not really suitable.

Zoletil would work, but is expensive.
